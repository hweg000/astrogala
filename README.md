# 🌌 AstroGala: Galería Interactiva para Bodas

AstroGala es una plataforma web moderna diseñada para capturar y compartir los mejores momentos de tu boda de forma inmersiva y elegante.

## ✨ Características Principales

- **🌐 Domo 3D Interactivo:** Visualiza todas las fotos de los invitados en una esfera en 3D que flota en el espacio.
- **🖼️ Galería Infinita (Modo Tótem):** Una vista optimizada para pantallas gigantes/kioscos que muestra una pared infinita de fotos con scroll omnidireccional.
- **📱 Carga Instantánea:** Los invitados pueden subir fotos y videos directamente desde su celular escaneando un código QR, sin instalar nada.
- **💬 Interacción en Vivo:** Sistema de comentarios y "likes" en tiempo real para cada momento compartido.
- **🎨 Estética Premium:** Fondo de galaxia animado, micro-interacciones suaves y diseño minimalista oscuro.

## 🛠️ Tecnologías

- **Framework:** [Astro](https://astro.build/) + React.
- **3D Engine:** [OGL](https://github.com/oframe/ogl) (WebGl).
- **Backend:** Node.js (Servidor de carga) + JSON Database.
- **Estilos:** Vanilla CSS con micro-animaciones dinámicas.

## 🚀 Instalación y Desarrollo

1. **Instalar dependencias:**
   ```bash
   npm install
   ```

2. **Iniciar servidor de desarrollo:**
   ```bash
   npm run dev
   ```

3. **Compilar para producción:**
   ```bash
   npm run build
   ```
